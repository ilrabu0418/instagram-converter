# 인스타그램 글/스토리 변환 시스템

이 폴더는 인스타그램 콘텐츠 변환을 위한 Claude Code 프로젝트입니다.

## 작동 방식

사용자가 입력하는 형식에 따라 적절한 시스템을 적용합니다:

### [글쓰기] 태그가 있는 경우
→ `글쓰기/SYSTEM_PROMPT.md` 지침 적용
→ 피드용 긴 글로 변환
→ 해시태그 15-20개 추천

### [스토리] 태그가 있는 경우
→ `스토리/SYSTEM_PROMPT.md` 지침 적용
→ 24시간 스토리용 짧은 글로 변환
→ 스티커/기능 추천

### 태그가 없는 경우
→ 내용을 보고 적절한 형식 제안
→ 또는 사용자에게 질문

---

## 핵심 원칙

### 반응 있는 글쓰기 5단계
1. **청중이 있다고 생각하고 쓰기** - 질문으로 마무리
2. **도움될 정보 공유** - 주소, 가격 등
3. **진솔한 이야기** - 인간미 있게
4. **스토리와 함께** - 단순 나열 금지
5. **컨셉에 맞게** - 일관된 톤앤매너

### 말투 가이드
- SNS는 '글'이 아니라 '말'
- 일상 언어 사용 (~요, ~해요)
- 이모지 적절히 활용
- 혼잣말 금지, 팔로워에게 말하듯

## 출력 예시

### 피드 글쓰기 출력
```

[본문]

#️⃣ 추천 해시태그
[해시태그 15-20개]

💡 변환 포인트
[적용된 원칙 설명]
```
