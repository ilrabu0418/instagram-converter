## 인스타그램 스토리 특징

### 스토리 vs 피드 차이점
| 구분 | 스토리 | 피드 |
|------|--------|------|
| 노출 시간 | 24시간 후 삭제 | 영구 보관 |
| 길이 | 짧고 임팩트 있게 | 길고 상세하게 |
| 목적 | 즉각적 반응, 실시간 소통 | 정보 제공, 기록 |
| 톤 | 가볍고 캐주얼 | 정돈된 느낌 |

## 스토리 유형별 템플릿

### 1. 일상 공유 스토리
```
[사진/영상 + 짧은 텍스트]
"오늘도 화력 풀가동 중🔥"
"퇴근길에 들러주세요~"
```

### 2. 질문/투표 스토리
```
[투표 기능 활용]
"오늘 뭐 드실래요?"
□ 껍데기
□ 삼겹살
```

### 3. 비하인드 스토리
```
[주방/준비 과정 공개]
"오늘의 숙성 상태 확인✅"
"이게 바로 30년 내공..."
```

### 4. 프로모션 스토리
```
[신메뉴/이벤트 알림]
"NEW! 닭목살구이 출시🐔"
"위로 스와이프해서 예약하기👆"
```

### 5. 고객 후기 리그램
```
[고객 사진 공유]
"@고객님 방문 감사합니다🙏"
"맛있게 드셨다니 보람있네요"
```

### 6. 실시간 현장 스토리
```
[라이브한 분위기]
"지금 자리 있어요! 빨리 오세요ㅋㅋ"
"오늘 예약 마감 임박⚡"
```

## 스토리 작성 원칙

### DO (해야 할 것)
- 짧고 임팩트 있게 (1-2문장)
- 이모지 적극 활용
- 스티커, 투표, 질문 기능 활용 제안
- 긴급함/희소성 표현 ("오늘만", "한정")
- 행동 유도 (CTA) - "스와이프", "DM주세요"

### DON'T (피해야 할 것)
- 너무 긴 텍스트
- 딱딱한 문어체
- 해시태그 남발 (스토리엔 불필요)
- 과도한 홍보 느낌
